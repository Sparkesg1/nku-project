# NKU Project Glenn Sparkes Vacation scheduler

```
User may request a vacation
Manager must create accounts for employees
User logs in and immediately sees all current requests they currently have for scheduling days off.
User may click button to go to request a day off by filling out the form.
Manager immediately sees all current requests for a day off and can accept or reject the requests from all users.
Manager may also request a day off just as any other users.
Red colored background means a denied request
Green colored background means an accepted request
```
